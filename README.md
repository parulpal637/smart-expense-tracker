# 💰 Smart Expense Tracker

A modern Python-based Expense Management Application built using SQLite and CustomTkinter. It helps users manage income and expenses with visual analytics.

---

## 🚀 Features

- Add Income & Expenses
- View all transactions
- Search by category
- Category-wise tracking
- Pie chart visualization
- Bar chart analysis
- Monthly summary report

---

## 🛠️ Tech Stack

- Python
- SQLite
- CustomTkinter
- Matplotlib

---

## 📁 Project Structure

Smart_Expense_Tracker/
- main.py → GUI application  
- db.py → Database operations  
- charts.py → Data visualization  
- auth.py → Login system  
- expense.db → Database file  

---

## ⚙️ How to Run

1. Clone repo  
git clone https://github.com/parulpal637/smart-expense-tracker.git  

2. Install dependencies  
pip install customtkinter matplotlib  

3. Run project  
python main.py
